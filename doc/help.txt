    A small cli for managing tasks in the terminal.

{_bold}Commands{/}:

    {green}${/} {_bold}[a]{/}dd [name]               Add a new task
    {green}${/} {_bold}[d]{/}elete [name]            Deletes a task
    {green}${/} {_bold}[c]{/}omplete [taskId]        Mark a task as completed
    {green}${/} {_bold}[u]{/}ncomplete [taskId]      Mark a task as incomplete
